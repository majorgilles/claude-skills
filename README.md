# claude-skills

A collection of [Claude Code](https://claude.com/claude-code) skills.

## Skills

- **coding-tutor** — hands-on coding tutor; the student types every line, Claude explains why
- **exercise-tutor** — assigns exercises with the recipe spelled out; student writes every line
- **find-skills** — helps discover and install agent skills
- **grill-me** — relentless interview to stress-test a plan or design
- **tdd** — test-driven development with red-green-refactor loop
- **to-issues** — break a plan into independently-grabbable GitHub issues
- **to-prd** — turn current conversation context into a PRD and submit as a GitHub issue

## Installation

Drop any skill directory into `~/.claude/skills/` (Linux/macOS) or `%USERPROFILE%\.claude\skills\` (Windows). Claude Code picks them up automatically.
